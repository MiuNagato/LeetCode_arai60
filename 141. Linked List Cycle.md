## Step2

### 同じ問題を解いた人のプルリクを見る

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):  # コンストラクタ
#         self.val = x
#         self.next = None

class Solution:
    # Optional[ListNode] は「ListNode か、もしくは None が入る」
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        visited = set()  # 重複NGのコレクション
        node = head
        while node:  # nodeがあるうちは繰り返す
            if node in visited:
                return True
            visited.add(node)
            node = node.next
        return False
```

## Step3

### 3回連続で再現

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        node = head
        visited = set()
        while node is not None:
            if node in visited:
                return True
            visited.add(node)
            node = node.next
        return False
```
